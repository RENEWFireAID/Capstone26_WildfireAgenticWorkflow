# LLM-APP

A lightweight, production ready retrieveal service designed to empower LLM-based wildfire research systems, 

The repository exposes a structured HTTP API that allows large language models (LLMS) to retrieve semantically relevant document chunks from a curated wildfire knowledge base. 

It is built to support: 
* Retrieval-Augmented Generation (RAG)
* Agent-based decision pipelines
* Structured and Secure API integrations
